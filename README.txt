Thiệp Sinh Nhật - Personalized Demo

Files included:
- index.html
- style.css
- script.js

How to use:
1. Upload these files to your hosting or GitHub repo root.
2. Open index.html — you will see the input form.
3. Fill name & message then click 'Tạo thiệp' to show the personalized card with effects.
Note: To use background music, upload your MP3 named 'birthday.mp3' into the same folder.
